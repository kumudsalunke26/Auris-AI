# AURIS AI – Advanced Unified Recognition & Intelligence System

AURIS AI is an AI-powered multilingual language processing platform that enables language detection, machine translation, sentiment analysis, speech-to-text (STT), and text-to-speech (TTS) through a unified interface.

## Features

* 🌐 Language Detection
* 🔄 Machine Translation
* 😊 Sentiment Analysis
* 🎤 Speech-to-Text (STT)
* 🔊 Text-to-Speech (TTS)
* 🌍 Supports 10+ Languages

## Technologies Used

* Python
* Gradio
* LangDetect
* TextBlob
* Deep Translator
* SpeechRecognition
* Edge-TTS

## Research Focus

AURIS AI explores multilingual Natural Language Understanding (NLU) by integrating multiple NLP tasks into a single platform. The project investigates language processing across diverse language families and aims to improve accessibility for multilingual users.

## Deployment

The application is deployed on Hugging Face Spaces for public access and scalable inference.
https://huggingface.co/spaces/kumudsalunke/Auris-AI
